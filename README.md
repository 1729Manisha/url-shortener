# URL Shortener

This is a simple URL shortener application that allows users to create short URLs for long URLs. The application has the following features:

- Generate a short URL for a given URL.
- Allow users to input custom short URLs.
- Display the original URL when given the short URL.
- Admin access to view all generated short URLs and their original URLs.

## Technologies Used:
- Python (FastAPI)
- DynamoDB (AWS)
- Pydantic for strong typing
- Uuid for random short URL generation
